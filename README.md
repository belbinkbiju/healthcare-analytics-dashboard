# Healthcare Analytics Dashboard

An interactive Power BI dashboard project designed to analyze healthcare encounters, patient behavior, insurance coverage, and operational trends using healthcare datasets.

---

# Project Overview

This project focuses on transforming raw healthcare datasets into meaningful business insights through data cleaning, modeling, visualization, and interactive dashboard design using Power BI.

The dashboard suite provides insights into:

* Healthcare encounter trends
* Insurance coverage and claim analysis
* Patient readmission behavior
* Procedure utilization patterns
* Operational and financial healthcare metrics

---

# Dashboard Pages

## 1. Encounters Overview

Provides a high-level operational overview of healthcare encounters.

### Key Insights:

* Total Encounters
* Average Encounter Duration
* Encounter Trends by Year
* Encounter Distribution by Class
* Encounter Duration Categories

---

## 2. Cost & Coverage Insights

Focuses on financial analysis and insurance coverage.

### Key Insights:

* Total Claim Cost
* Insurance Coverage Analysis
* Average Procedure Cost
* Payer-wise Claim Trends
* Top Procedures by Average Cost

---

## 3. Patient Behavior Analysis

Analyzes patient engagement and readmission trends.

### Key Insights:

* Unique Patients
* Readmission Rate
* Average Visits per Patient
* Patient Trends Over Time
* Most Common Procedures

---

## 4. Encounter Class Deep Dive

Interactive Drill-Through page for detailed encounter class analysis.

### Features:

* Dynamic filtering
* Drill-through navigation
* Dynamic DAX titles
* Contextual patient and procedure insights
